<objective>
Add `project.url` to the Linear snapshot pipeline (D-13) so the timeline popover can
link to the authoritative Linear project URL. Linear URLs use a slug-id (not the raw
UUID) and cannot be reconstructed client-side, so the field must be fetched from the API
and carried through the sanitized snapshot pipeline.

Purpose: TL-03 requires a Linear link in the milestone/project popover. This plan makes
`url` available in `public/roadmap.json` (populated later by the gated snapshot re-run in
04-07) and validates it as PII-safe.
Output: `url` threaded through query -> map -> transform -> schema, with url-passthrough
and assertNoLeak-URL tests; all field additions are additive and backward compatible.
</objective>
