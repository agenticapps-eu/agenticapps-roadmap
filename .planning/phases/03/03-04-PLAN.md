| Boundary | Description |
|----------|-------------|
| browser → /api/linear/snapshot | Live fetch crosses to the proxy (gated by Access at the edge) |
| live response → UI | Already leak-gated server-side; client re-validates shape only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-10 | Information Disclosure | malformed/hostile live response | mitigate | Client re-validates with the SAME RoadmapJsonSchema; any mismatch → fall back to snapshot, never render unvalidated data (REQ-LOADER, ASVS V5). |
| T-03-11 | Denial of Service | live failure crashing the app | mitigate | Entire live branch in one try/catch → never throws; falls back to snapshot + notice on ANY failure mode. Genuine snapshot outage still hits RoadmapError. |
| T-03-12 | Information Disclosure | token reaching the client | mitigate | Loader fetches the proxy, never Linear directly; no token in the bundle (re-confirmed by the phase-gate `lin_api_` grep). |
| T-03-SC | Tampering | npm/pnpm installs | accept | No installs in this plan. |
