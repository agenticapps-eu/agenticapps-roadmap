|----------|-------------|

| Pages Function → GitHub Actions REST | dispatch/status carry the PAT (server-side only) to trigger + read backfill runs. |
| CI runner → sibling private repos + Linear | the PAT (Contents:Read) clones siblings; LINEAR_API_KEY writes Linear. |

Proven by Task 3's curl checks (302/403 on all four paths) — UAT items 2 & 8. A `200` from any unauth path FAILS the phase. |
| T-08-07 | Elevation of Privilege / Tampering | replay of a `previewRunId` to authorize an unreviewed apply (CR-01) | mitigate | 15-min recency bound (Phase 7) + KV consume-once nonce (plan 08-01), verified live in Task 3 item #13 (reused id → 403). |
